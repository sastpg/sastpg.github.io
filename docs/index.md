!!! info "Introduction"
    你好，欢迎来到我的小站！以下是关于这个网站稍显凌乱的介绍。

    计算机科学的知识和技能纷繁复杂，且更新迭代很快，很容易造成学习完一项技术后一段时间内没复习就忘掉的窘境。因而我便想将自己的一些笔记整理归类，希望将知识和技能体系化。于是创建个人知识库的想法就在我的脑海中诞生了。目前我对它的定位就是──用于构建自己的知识体系并且记录一些思想零碎。

    **愿我们的生活在一次次偶然的交汇后，都能有所沉淀！**
